IMPORTING AN ULTRABAC LICENSE

There are two methods of importing an UltraBac License:

  • Save File method
  • Copy & Paste method

License keys are encrypted, and can be read only by UltraBac's licensing module. Please note you MUST designate a machine to be the UltraBac License Server when installing the software on multiple machines in one location.
 

SAVE FILE METHOD:

  1. Save the attached text file (UBLicense.txt) to the UltraBac install directory. For example, the default path is "C:\Program Files\UltraBac Software\UltraBac9" for UltraBac Version 9.x.
  2. Open the UltraBac Management Console.
  3. For Version 9.x click 'Manage\Licensing\Import License.' For previous versions of UltraBac, click 'Tools\License Management\Import License.'
  4. The Radio Button for 'Import license from File' should already be selected.
  5. Click 'OK.'
  6. Select the UBLicense.txt file.
  7. Click 'Open.'

COPY & PASTE TEXT METHOD:

  1. Open the UBLicense.txt file.
  2. Copy all of the license key characters into Clipboard.
  3. Open the UltraBac Management Console.
  4. For Version 9.x click 'Manage\Licensing\Import License.' For previous versions of UltraBac, click 'Tools\License Management\Import License.'
  5. The Radio Button for 'Import license from File' should already be selected.
  6. Click 'OK' to complete the licensing operation.
