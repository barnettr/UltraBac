UltraBac SoftwareはVMworld 2008で市場初の機能を発表

--　ライブシステムのバックアップと同時にディザスタリカバリ（復旧）のVMDKファイルを作成する機能を提供　--

 

ネバダ州ラスベガスおよびワシントン州ベルヴュー市 - (2008年９月１８日) - UltraBac Software社は、稼働システムをバックアップしながら、同時にVMware ESXやGSXの仮想マシンディスクフォーマット(VMDK)ファイルを一工程で同時に作成するUBDR Goldの新しい機能を発表しました。 通常のイメージバックアップ中にESXやGSXのVMDKファイルを作成する機能は市場初です。この新しい機能は、通常の物理システムから仮想システム(P2V)への変換を不要にすることでディザスタリカバリを非常に簡素化します。 この新しい機能はユーザフレンドリーなVMware Consolidated Backup (VCB)エージェントとともに今年の第４四半期にリリースされる予定です。

この新しい機能のあるUBDR Goldを使用すると、システム管理者はイメージファイル、ESX VMDKファイルやGSX VMDKファイルのシングルバックアップジョブを作成することができます。 VMDKファイルを選択した場合、そのセットのバックアップパスを介して自動的にアップロードすることができます。VMDKファイルはバックアップによって既に作成されているため、 P2Vディザスタリカバリの作業を最小限の時間で行うことができます。物理サーバのUBDR Goldバックアップによって作成されたVMDKファイルは直接アップロードされるため、ファイルは必要な時に早急に起動に使用することができます。 この機能の別の独自の特徴は、差分および増分バックアップの両方がベースのVMDKファイルを自動的に更新することができ、あらゆるP2V作業に対して常に最新の状態であるということです。 ディザスタリカバリの時間を最小限にするためには、管理者は、この「予備の」仮想マシンを作成するだけです。

UBDR Goldで保護されたサーバが故障した場合、オペレーターは以下の２つの方法でディザスタリカバリ（復旧）を実行することができます。 （１）UBDR Goldのイメージベース・ディザスタリカバリ機能を使用して物理サーバを早急に復旧する。または、（２）仮想環境で故障したサーバを復旧し、さらに高速でマシンをオンラインに戻す。 仮想リカバリを完了するために必要なことは、ユーザは予備の仮想マシンを起動するだけです。この新しい機能は、日々のバックアップ計画の一部としての直接搭載可能なディザスタリカバリ・ソリューションを提供することによって、 VMware管理者にシステム・アップタイムを安価に満たすまたは上回ることを可能にします。

この機能の登場前は、P2V作業を行うには多くのステップが必要であり、その間、エンドユーザ環境における全てのディスクやパーティションタイプの組み合わせには対応していないため変換はたやすく妨げられる可能性がありました。 UBDR Goldは、通常のバックアップの一部として.vmdkファイルを作成するため、変換は必要なく余分な時間も掛かりません。UBDR Gold によるVMDKファイルバックアップは、全てのディスクタイプ(Basic、DynamicおよびGPT)、 全てのパーティションタイプ(Primary、Extended、LogicalおよびSimple volume)および全てのWindowsファイルシステムタイプ(FAT16/32、NTFSおよびEISA)へ書き込むことができます。これにより、 高い柔軟性を提供し、従来のP2V変換を事実上不要にし、.vmdkファイルを物理ディスクのように例外なく読み取ることができるため、システム保護を本質的により確実にします。

副次的な利点として、システムバックアップを仮想環境で保存することは、１つの物理ディスクに複数の仮想マシンを保管できるため「グリーン・データ・センター」の実現に役立ちます。

UltraBac SoftwareのCEO兼創業者であるMorgan Edwardsは、「弊社は、バーチャル・マーケットにおいて市場初のバックアップおよびディザスタリカバリ機能の提供できることを誇りに思っています。」と述べました。 「VMware Select Partnerとして、弊社は、日々のUBDR Goldバックアップを手軽に使用した常に準備万端の状態にあるP2Vディザスタリカバリ・ソリューションの構築を目指しています。弊社のお客様は、 日々の差分または増分バックアップを用いてVMDKファイルを更新できるようにした弊社開発チームの努力を特に評価していると思います。この新しい機能では、P2Vベースのディザスタリカバリの開始から終了までに必要な時間は、 仮想マシンをオンラインにさせる時間だけです。このディザスタリカバリのプロセスは非常に簡単であり、また購入しやすいです。」

入手について
P2V、P2P、V2PおよびV2V機能を含むVMware対応のUBDR Gold v4.0は現在販売中です。新しいVMDK機能およびVirtual Consolidated Backup Agent は２００８年の第４四半期に入手可能です。

UltraBac Softwareについて
UltraBac Softwareは、信頼性のある、高速で革新的なバックアップおよびディザスタリカバリ・ソフトウェアをあらゆる規模の企業に提供し評判を得ています。 UltraBacファイルバックアップおよびUBDR Goldイメージバックアップは、弊社の主力製品です。UltraBacは、広範囲のオプションと組み込みの柔軟なスケジュール機能を備えたファイル毎のバックアップおよび復元を提供します。 Exchange、SQL、SharePoint、Locked File、OracleおよびTivoli などのオプションエージェントは、主力製品に拡張した機能を提供します。ベア・メタル・リカバリで（BMR）は、UBDR Gold はイメージベースのバックアップおよびディザスタリカバリを実行し、故障した物理サーバをわずか６分で復元できます。バックアップはテープ、ディスク、UNCパス、SAN/NASデバイスおよびIBM TSMに実行できます。 UBDR Goldは、異機種ハードウェア間での復元および変換に完全に対応し、また、P2P、P2V、V2VおよびV2P変換やディザスタリカバリ操作に高い柔軟性を提供します。トライアルソフトウェアや詳細情報に関しては www.ultrabac.com を参照してください。
