UBDR Pro: スナップショットに基づいた災害復旧

 

重要なWindowsサーバーを15分以内で復旧!　広範な実績のあるUltraBac SoftwareのUBDR Proを使用すれば、たとえば起動できなくなったマシンのOSパーティションが4GBであっても、イメージベースの災害回復技術を使用して、すぐに容易に復旧できます。標準的な復旧技術を使用すると、故障したサーバの回復に2時間から2日くらいかかる可能性があります。しかし、UBDR Proのイメージ処理は、全過程を、専門知識がなくても誰もが重要なマシンの災害回復をすぐに容易に実行できるような、簡単で合理化された処理にします。

 

UltraBac の災害復旧ソフトウェアであるUBDR Proは、イメージ (「スナップショット」とも呼ばれます) に基づいたバックアップと災害復旧の技術を使用して、Windows サーバーやワークステーションの使用者に、軽快で速い復元を提供します。このソフトウェアは、Windows NT 4.0、2000、2003、XP を搭載したサーバーやデスクトップマシン上で動きます。UBDR Proの使用するUltraBacのイメージバックアップ実行機能は、Windowsベースの汎用起動CD環境といっしょに動作して、起動できる OSが存在しないマシンでも、1つまたは複数のディスクパーティションを復元します (小容量のディスクを大容量のディスクに変更することもできます)。UBDR Pro のライセンスは、単独での使用も、ファイルごとのバックアップと復元用に広く使用されるUltraBacソフトウェアに完全に統合しての使用も可能です。これは、通常のファイルごとのバックアップを実行する他社製品を使っていても、災害復旧用のUBDR Proが使用できることを意味します。

 

革新的な技術を使用するUBDR Proは、予期しないコンピューターダウンタイムに関連する不適当な金銭的な損失を防ぐ、組織の第一の防衛線になるように設計されています。導入後は、自動スケジューラが、選択されたパーティションのスナップショット (正確なイメージ) を、ローカルのテープ、ディスク、任意のネットワーク(UNC)パスへ初期化します。組み込みのLocked File Backupエージェントは、ファイルが全て安全にバックアップされ、要求されたときにはそれらを完全に復元できることを保証します。このUltraBacイメージエージェントはパーティション内の使用中のディスククラスタだけをバックアップし、不適切なページファイルや休眠データを無視します。また、バックアップは、ネットワーク上の記憶装置への転送前にクライアント側で圧縮されます。これらの機能は、媒体の必要容量を減らして、バックアップの速度と復元の速度の両方を向上させます。その結果、トラフィックが軽減されると同時に、DRバックアップ処理全体の全面的なスループットは増大します。

 

その軽快で速い転送速度により、UBDR Pro は故障したシステムを回復する標準的なファイルごとのバックアップと復元の方式では、マシンの再構築に、通常は数時間から数日かかるところを、数分で復元できます。UBDR Pro では、たとえば高速なギガビットネットワーク上または LTO2 テープ使用時には、1分あたり 1000MBの復元が可能です。復元を開始するには、使用者は単に汎用 UBDR Pro 起動CDを挿入するだけです。その他の復元手順も、特別なITの訓練や知識、経験も要りません。スナップショットの復元後、障害の発生したマシンは、迅速な再起動の後で最新のイメージバックアップの状態に 100% 復元されます。

 

UBDR Proの副次的な特長は、サーバー複製能力です。いちから新しいマシンを構築するときに、UltraBacのイメージ技術は非常に有用です。従来の構築方式では多くの時間と資源が必要となるのに比べて、UBDR Proでは、使用者はUBDR Pro CDから起動することにより、自分のネットワークから指定の「template (ひな形)」をコピーして、全く同じ内容の新しいマシンを数分で作成できます。

 

UBDR Pro は、賞を受けた組み込み暗号化 [1] 機能も提供しているため、 バックアップ媒体の保管場所が自社内でも自社外でも、政府その他の機密保護の要請に完全に応じることができます。 UltraBac の提供する組み込みの機密保護機能は、極秘データのプライバシーを、それがネットワークや選択されたバックアップ媒体へ転送される前の段階で保護する機能を提供します。今日では銀行や保険会社などの多くの企業が、法律 (米国ではGramm-Leach-Bliley Act : 米国金融制度改革法) により、新しいプライバシー法に則った極秘の金融情報保護を要求されています。この目的のために、UltraBacは最長127文字の暗号化句 (フレーズ) を使用してデータを暗号化し、顧客が1016ビットの暗号鍵を使用できるようにします。この機能は、顧客が必要に応じて暗号化保護の量を変更することを可能にし、追加費用のかからない強力なデータ機密保護を提供します。

 

回収はすぐです。1つの災害または 1つのマシンの障害だけでも、ROI (費用対効果) 要因はすぐに100:1以上になります。企業は、サーバーまたはシステムのダウンタイムの費用を、1時間あたり5,000ドルから1分あたり15,000ドルまでの範囲に見積もります。UBDR Pro を持つことは、そのような予期しない危険に対する保険に加入することに似ています。

UBDR Pro がどのように動作するかの詳細:

ディスクまたはテープのどちらかに「実際の」イメージのバックアップを供給します。イメージバックアップは、ディスク上の単一またはすべてのパーティションの、ある時点でのスナップショットや「静的なミラーのコピー」と同じです。 バックアップは1つ以上のセットと1つのグループで構成されます。1つのセットは、1つのディスク上の1つ以上のパーティションの定義です。グループは、バックアップ対象のすべてのセットと、すべての制御パラメータ(開始時間、消去メディア、削除メディア、電子メール通知、UNCのパスまたはテープ装置番号その他)を含みます。バックアップは、GUIから対話的に実行することも、

(a) スケジューラのサービスから自動実行されるように予定を設定すること、
(b) 完全にスクリプト化 (100%コマンド行駆動) してエンドユーザの任意のアプリケーションに統合することも可能です。バックアップ時に開かれているすべてのファイルの完全性を保証するために、UltraBacの選択肢にあるLocked File Backup (ロックされたファイルバックアップ) エージェントが組み込まれています。この機能は、直接ディスクを読む低水準のバックアップを実行するため、OSは使用しません。ディスク上のクラスタのうち使用中の印のあるものだけをバックアップして、実行されるバックアップには、不必要なものや空のディスク空間はいっさい含まれません (結果的に非常に高速です)。災害復旧操作の一部としての回復は、障害が発生して起動できなくなったマシンの復旧か、以前のある時点へのマシンの復旧回復 (たとえば、新しいクラスの開始前にトレーニングマシンを復旧回復) を、第一に意図します。新しいバックアップのパラメータは、ディスクまたはテープのどちらかに格納されたイメージバックアップから、個々のファイルを復旧する能力を使用者に供給します。イメージの回復は、起動できなくなったマシンの復旧や、より大きなディスクへのアップグレードを、システム管理者が簡単にできるようにします。障害が発生したマシンを復旧する全過程は、単一の、汎用の起動CD (UBDR Proが対象としている全マシン用のCDと同じもの) を挿入し「電源」ボタンを押すことにより完全に自動化できます。中央の管理者は設置配備および導入を完全にスクリプト化することが可能で、遠隔のUBDRを実行することもできます。UBDR起動処理はマシンを初期化し、バックアップソフトウェアを読み込み、ネットワーク接続を確立して、自動化されたOSパーティション回復処理や、十分な知識を持つシステム管理者によるその処理を可能にします。ネットワークの速度や復旧するディスククラスタ数に依存しますが、故障したマシンの2GBから4GBのOSパーティションのイメージは、任意の地点からのネットワーク接続経由で、10分から30分の間で回復を実行できます。
