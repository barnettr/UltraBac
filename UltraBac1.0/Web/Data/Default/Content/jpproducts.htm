UBDR Gold: 訳文 - ベアメタル　リカバリ

UBDR Goldの革新的な技術が可能にした圧倒的なフレキシビリティで、Windowsシステムをいつでも、どんなデバイスからでも、ほぼ全てのハードウェアに復元を行うことができます。

UBDR Goldの特徴＆利点

  • SNAPSHOT Windows LIVE!!! オンラインでOSイメージバックアップ
  • SNAPSHOT Windows LIVE!!! オンラインで全サーバーのイメージバックアップ
  • SNAPSHOT Windows LIVE!!! オンラインでクライアントのイメージバックアップ
  • Windows サーバーイメージバックアップデータを別のハードウェアへ復元
  • 物理システムから物理システムへ(P2P) -　Dell　からIBM、IBMからCompaqなど、その他さまざまな移行
  • Windowsサーバーイメージバックアップデータを仮想環境へ復元
  • 物理システムから仮想システムへ（P2V）-　VMwareやMicrosoft　仮想サーバーに対応
  • Windowsサーバーイメージバックアップデータを仮想環境に復元
  • 仮想システムから仮想システムへ（V2V） - VMwareからMicrosoft Virtual Serverへ,またはその逆へ。
  • Windowsサーバーイメージを物理環境に復元
  • 仮想システムから物理システムへ（V2P）-　VMwareイメージをDELL1850へ、およびその他の移行
  • UBDR Gold　Recovery メディア(WinPEベース)によるコールドブートバックアップ
  • 無人でのサーバー復元処理　-　100％スクリプト化されたリカバリプロセス
  • 差分および/または増分イメージバックアップ
  • ファイルバックアップの実行とWinPE起動環境からの復元
  • SFTPデバイス対応を追加 **
  • Tivoliストレージマネージメント（TSM）デバイス対応を追加 **
  • UBDR Goldフルイメージバックアップがサポートする単一ファイル復元（SFR)
  • Dell DRAC, HP iLO,および起動を可能にするPXEを使用したリモートサーバー復元機能
  • Itanium IA64 およびx64 Windowsオペレーティング環境 のネイティブサポート
  • アクティブセクタのみをバックアップして必要なストレージ容量を最小化
  • ページファイルとハイバネーションファイルを無視して必要なストレージ容量を最小化
  • パーティションを個別に選択またはスナップショットイメージバックアップで一度に全てを選択
  • イメージバックアップを自動的に圧縮
  • UBDRの汎用性の高いISOファイルでどのWindowsサーバーも起動
  • テープ、ディスク、またはネットワークUNCパスから復旧
  • 最大６GB/分の復元速度
  • 簡単操作でパーティションのサイズ変更とデータ復元
  • Windowsの動作中に非アクティブなパーティションを復旧
  • リモート監視機能内臓
  • 追加されたデータのセキュリティ要求に対応する暗号化
  • 動作前後にVBSや他のプログラムを使用する機能
  • バックアップの成功/失敗をチェックする集中型リポートコンソールの追加
  • リモートストレージへバックアップを移行するUltraCopy **
